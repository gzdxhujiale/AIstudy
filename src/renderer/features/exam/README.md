# 考试模块

## 范围

考试模块提供第一版闭环：题库录入、当前知识文档提题、题库 JSON 导入导出、整套试卷编写、考试作答、提交后自动评阅、成绩记录查看。

## 参考来源

业务模型参考 `SnowLove0303/Knowledge-base` 仓库 `quiz-system` 分支：

- 题库：题干、选项、答案、分类、难度、分值。
- 试卷：试卷元信息与题目顺序。
- 考试记录：试卷、得分、答题明细；当前版本不要求填写考生名。
- 自动评阅：客观题按答案匹配，提交后立即保存成绩。

当前项目是 Electron + React，未照搬 PyQt 界面。考试数据通过 Electron 主进程写入公开版固定 MySQL 表；MySQL 不可用时先保存 IndexedDB 本地快照，恢复后由页面加载/保存流程同步。

## 对接状态

- 已接入主导航的当前课程上下文。
- 题目、试卷、考试记录都带 `courseId` 与 `courseName`，页面只显示当前课程下的数据。
- 旧版未归属考试数据会在打开考试模块时归入当前课程，避免升级后看不到数据。
- 题库支持从当前选中的知识库文档导入结构化题目。
- 题库支持 JSON 导入导出，导出的题库文件可再次导入。
- 试卷支持按“大题/分区”编辑整套卷子，旧版只保存题目顺序的试卷会自动迁移到默认分区。
- 考试页支持以试卷卡片选择整套试卷，作答和预览按分区展示。
- 参考 `2026年考研政治真题试卷.pdf` 的扫描版结构，内置 `2026年考研政治真题试卷`：已写入本地文字版 1-33 道客观题和 34-38 道材料分析题；客观题支持自动评阅，材料分析题保留参考答案。

## 导入格式

知识库文档导入会识别按题号分段的结构化文本：

- 题干行：`1. 题干内容` 或 `题目：题干内容`。
- 选项行：`A. 选项内容`、`B. 选项内容`。
- 答案行：`答案：A`、`答案：AB`、`答案：正确`、`答案：参考文本`。
- 可选行：`题型：单选/多选/判断/简答`、`分类：分类名`、`分值：5`、`难度：1`、`关键词：词一,词二`。

JSON 导入支持数组或 `{ "questions": [] }` 结构，字段可使用 `stem/question/题干/题目`、`type/题型`、`options/选项`、`optionA/选项A`、`answer/答案`、`keywords/关键词`、`category/分类`、`score/分值`、`difficulty/难度`。

## 文件

- `ExamWorkspace.tsx`：考试页面、题库、试卷、作答、成绩 UI。
- `examImportExport.ts`：题库 JSON 导入导出、知识文档文本提取、结构化题目解析。
- `examService.ts`：MySQL 优先/本地兜底的数据读写、试卷分区模型规范化、自动评阅。
- `politics2026Seed.ts`：2026 考研政治真题种子数据。
- `examTypes.ts`：考试模块类型定义。

## 评阅规则

- 单选、判断：答案完全一致得分。
- 多选：选项集合完全一致得分。
- 简答：标准答案完全匹配，或配置的关键词全部命中得分。

## 数据库

Renderer 不直接访问 MySQL。`electron/main.ts` 通过 `exams:load` / `exams:save` IPC 维护固定表：

- `exam_questions`
- `exam_papers`
- `exam_paper_sections`
- `exam_paper_questions`
- `exam_attempts`
